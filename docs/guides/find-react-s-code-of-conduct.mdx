# Find React's Code of Conduct

<Callout type="info">
Locate and access React's Code of Conduct from the React Foundation website.
</Callout>

### Step 1 — Click the Learn link

Click the **Learn** link in the main navigation. This takes you to the learning resources section of the React Foundation website.

### Step 2 — Click the Reference link

Click the **Reference** link. This navigates you to the reference documentation area where community guidelines are organized.

### Step 3 — Click the Community link

Click the **Community** link in the Legacy APIs section. This section contains links to community-related resources and policies.

Click the **Code of Conduct** link in the News section. This opens the full Code of Conduct document that outlines community standards and expectations.

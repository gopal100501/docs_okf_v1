# View Agentforce Features in Salesforce Go

<Callout type="info">
Access the Agentforce features overview in Salesforce Go from the Administration section of Setup.
</Callout>

### Step 1 — Open Setup Home

Click the **Setup Home** link in the Administration section. This takes you to the main setup interface where you can manage your Salesforce organization.

### Step 2 — Navigate to Salesforce Go

Click the **Salesforce Go** link. This opens the Salesforce Go configuration area where you can set up and manage mobile and agent features.

<CaptureImage src="./assets/view-agentforce-features-in-salesforce-go/step-02.jpg" alt="Step 2" render="eyJoaWdobGlnaHQiOnsieCI6MC4wMTQzMjQwODIzNjM0NzM1OSwieSI6MC40NDAxMTU3NDA3NDA3NDA3LCJ3aWR0aCI6MC4wODY0NzYwNTE5MjQ3OTg1NywiaGVpZ2h0IjowLjAyMjIyMjIyMjIyMjIyMjIyM319" />

### Step 3 — Select the Initial Setup tab

Click the **Initial Setup** tab in the Salesforce Go section. This displays the setup options available for configuring Salesforce Go.

<CaptureImage src="./assets/view-agentforce-features-in-salesforce-go/step-03.jpg" alt="Step 3" render="eyJoaWdobGlnaHQiOnsieCI6MC4zMTA0NDM3MTA4MzI1ODcyNiwieSI6MC40NjM4MTk0NDQ0NDQ0NDQ0NSwid2lkdGgiOjAuMDY5Mzk2MjYyMzA5NzU4MjcsImhlaWdodCI6MC4wNTkyNTkyNTkyNTkyNTkyNn19" />

### Step 4 — View Agentforce features

Click the **Features** tab under Get Started With Agentforce. This shows the available Agentforce features you can enable and configure for your organization.

## Summary

You are now viewing the Agentforce features available in Salesforce Go. From here you can review and configure the features for your setup.

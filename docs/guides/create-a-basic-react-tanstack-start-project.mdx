# Create a Basic React TanStack Start Project

<Callout type="info">
Open a React TanStack Start basic example in CodeSandbox and explore the project documentation.
</Callout>

### Step 1 — Open the example in CodeSandbox

Click the **CodeSandbox** button in the React Example: Start Basic section. This launches the starter project in CodeSandbox, where you can view and edit the code in your browser.

### Step 2 — Close the dialog

Click the **Close dialog** button to dismiss any open modal or overlay. This clears the view so you can work with the project.

<CaptureImage src="./assets/create-a-basic-react-tanstack-start-project/step-02.jpg" alt="Step 2" render="eyJoaWdobGlnaHQiOnsieCI6MCwieSI6MCwid2lkdGgiOjAuOTgzNzQzNTUyMzkzNTYxMywiaGVpZ2h0IjoxfX0=" t="2467" />

### Step 3 — Navigate to the home page

Click the **Home** link in the documentation navigation. This takes you to the main documentation page where you can access other sections like Get Started, Tutorial, Guides, and Examples.

<CaptureImage src="./assets/create-a-basic-react-tanstack-start-project/step-03.jpg" alt="Step 3" render="eyJoaWdobGlnaHQiOnsieCI6MC4xMDE3NjE3NTE3OTM2NDE2NCwieSI6MC4wODAyNDY5MTk0NjcxMjU0MSwid2lkdGgiOjAuMDM4MjA4NDMwMjQ1NDE4NjgsImhlaWdodCI6MC4wNjE3Mjg0MDA5NDg2MDY4OX19" t="4583" />

### Step 4 — Switch the display theme

Click the **Theme: Auto. Switch to light mode.** button in the top navigation. This toggles the interface theme between light and dark modes to suit your preference.
